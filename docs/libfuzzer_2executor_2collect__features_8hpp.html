Calculate features of specified execution result. "Feature" is a outstanding feature of the execution which has unique ID. In most case, entering a new edge that is not covered by previous executions is a feature. "Features" is a vector of feature. libFuzzer calculate weight of the execution result that affects by features. If ChooseRandomSeed is using non-uniform distribution, input of higher weighted execution result is selected more frequentry.
