Detailed Description

template<typename T, typename U, typename Enable = void>
class fuzzuf::utils::range::shared_iterator< T, U, Enable >

Iterator adaptor that holds value of U and forward all operations to iterator of type T value of U is copied when this iterator is copied The main purpose of this adaptor is to keep smart pointer to the container during at least one iterator is alive.

example: std::shared_ptr< std::vector< int > > a{ new std::vector< int >{ ... } }; b = a|shared; b behave as same as *a, and keep reference count of a to be incremented until b is destroyed
