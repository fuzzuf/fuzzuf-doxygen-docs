Detailed Description

template<class I, class O, bool IS_REGULAR = true>
class fuzzuf::hierarflow::HierarFlowNode< I, O, IS_REGULAR >

The wrapper of HierarFlowNodeImpl. In order to support operators on HierarFlowNodeImpl, HierarFlowNodeImpl should be always handled via std::shared_ptr. This means we should not expose HierarFlowNodeImpl itself to users because it may result in using HierarFlowNodeImpl without std::shared_ptr. Therefore this is the class actually the instance of which users can create.

Template Parameters
IS_REGULARFuzzuf has special nodes, such as CallRandomChild, that act differently from usual nodes. For example, CallRandomChild randomly calls only one child instead of calling their children one by one. Also, it's possible that you and we will implement more other special nodes in the future. Because it would be confusing if we treat such nodes in the same way as usual nodes, we need to separate their type and the notation to use them.
