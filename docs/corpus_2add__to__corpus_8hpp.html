Insert execution result to corpus if that has features Unlike executor::AddToCorpus, this function always append the execution result to the corpus. This function puts id and sha1 hash on the execution result. If the execution result doesn't have filename and persistent is true, the sha1 hash is used as the filename.

Corresponding code of original libFuzzer implementation https://github.com/llvm/llvm-project/blob/llvmorg-12.0.1/compiler-rt/lib/fuzzer/FuzzerCorpus.h#L208
