Detect sequence of '0' to '9' characters and parse it as integer value, then apply one of following operation and serialize modified value to original position.

  • increment
  • decrement
  • multiply by two
  • divide by two
  • replace by random value

Corresponding code of original libFuzzer implementation https://github.com/llvm/llvm-project/blob/llvmorg-12.0.1/compiler-rt/lib/fuzzer/FuzzerMutate.cpp#L349
