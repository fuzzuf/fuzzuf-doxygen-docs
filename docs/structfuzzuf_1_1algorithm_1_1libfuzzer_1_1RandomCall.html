Detailed Description

template<typename F, typename Path>
class fuzzuf::algorithm::libfuzzer::RandomCall< F, Path >

Invoke randomly selected one of the child nodes. If the node doesn't have any child nodes, nothing are invoked. This node modifies flow. The node takes 1 path for RNG.

Since common random call implementation located in HierarFlow/Utility.hpp doesn't have ability to select RNG instance, this libFuzzer implementation cannot replace RandomCall to common version. In this implementation, RNG algoritm must be selectable, due to some tests require RNG that produce deterministic values.

Template Parameters
FFunction type to define what arguments passes through this node.
PathStruct path to define which value to to use.
