Detailed Description

Key-value store that bridges between fuzzers and optimizers.

Note
This class is implemented with the singleton pattern because we want to share Store instance between fuzzers and optimizers with no difficulty. This design gets devastating if we would like to have mutiple fuzzer instances in a single process/thread, but we currently assume that such situations don't happen.
