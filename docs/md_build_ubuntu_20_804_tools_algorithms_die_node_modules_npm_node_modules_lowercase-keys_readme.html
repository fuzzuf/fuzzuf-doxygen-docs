lowercase-keys

Lowercase the keys of an object

Install

$ npm install --save lowercase-keys

Usage

var lowercaseKeys = require('lowercase-keys');
lowercaseKeys({FOO: true, bAr: false});
//=> {foo: true, bar: false}

API

lowercaseKeys(object)

Lowercases the keys and returns a new object.
