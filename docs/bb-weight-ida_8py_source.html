17 # This script computes a weight value for each basis block of each functions. the algorithm is:
18 # 1. for each outgoing edge (i->j) of a BB i, assign a equal probability Eij, i.e. Eij= 1/n for a "n edges" BB.
19 # 2. assign root BB a weight of 1 (this is always reachable).
20 # 3. for each BB j, its weight is: W(j) = SUM (over i \in Pred(j)) W(i)*Eij
21 # after completion, it creates a pickle file that contains weights of BBs.
