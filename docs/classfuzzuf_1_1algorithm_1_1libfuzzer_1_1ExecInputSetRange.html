Detailed Description

template<bool unwrap, ExecInputSetRangeInsertMode mode>
class fuzzuf::algorithm::libfuzzer::ExecInputSetRange< unwrap, mode >

A range adaptor to make ExecInputSet to C++ range compliant.

Template Parameters
unwrapIf true, dereferenced value is range of std::uint8_t. Otherwise, dereferenced value is reference to ExecInput.
modeSpecify how to insert value.
