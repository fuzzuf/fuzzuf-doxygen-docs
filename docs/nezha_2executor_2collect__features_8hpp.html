Calculate features of specified execution result. Unlike CollectFeatures in libFuzzer, this function consider every features detected on the execution as unique_features of the execution. (This means unique_features is no longer unique. As the Reflection of that difference, original Nezha implementation renames unique_features to features.)

Corresponding code of original Nezha implementation https://github.com/nezha-dt/nezha/blob/master/Fuzzer/FuzzerLoop.cpp#L433
