90 // Define DEBUG_ASSERT, which is enabled only when the build mode is "Debug".
91 // NOTE: We know assert can be enabled and disabled already with NDEBUG.
92 // But, NDEBUG can be set/unset by CMake or other build systems with stealth.
93 // Due to this fact, writing "assert()" everywhere may get on our nerves
94 // becuase we would be worried if assert might be enabled even in Release mode.
