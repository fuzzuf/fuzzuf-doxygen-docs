void fuzzuf::algorithm::vuzzer::VUzzerMutator::ChangeRandomFull ( )

When fuzzer has full_bytes_dict, then insert bytes chosen from it at the offset. If not but has unique_bytes_dict, then insert bytes chosen from it at the offsets. Otherwirse replace bytes at offsets with random bytes generated from all_dicts. The offsets are chosen from taint information.

◆ TaintBasedChange()

void fuzzuf::algorithm::vuzzer::VUzzerMutator::TaintBasedChange ( )

Replace each bytes at offsets with random bytes. The offsets are chosen from taint information.

Replace random four bytes at offsets chosen from taint information about LEA operations with hard-coded magic bytes. Then replace random bytes at offsets chosen from taint information about CMP operations with magic bytes.
