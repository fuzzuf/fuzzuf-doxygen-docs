Insert execution result to corpus if that has features Note that, CollectFeatures must be applied to the execution result ahead, or the feature count is left to 0, and added result will never be selected by ChooseRandom Seed.

Corresponding code of original libFuzzer implementation https://github.com/llvm/llvm-project/blob/llvmorg-12.0.1/compiler-rt/lib/fuzzer/FuzzerLoop.cpp#L525
